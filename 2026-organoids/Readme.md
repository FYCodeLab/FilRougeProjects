# Fil Rouge "3D Organoids "Project — File list

This folder contains all datasets, documents, and materials related to the Fil Rouge programme at SupBiotech. 

Current content includes the 2026 project **“Design of a Semi-Automated 3D Printed System for Organoid Culture”**, with data, CAD files, protocols, and quality-control material. All files are provided to support transparency, reuse, and reproducibility in teaching and research.
